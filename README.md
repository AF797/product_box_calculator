# product_box_calculator
-------

## 설명
휴학하고 회사에서 아르바이트를 하던 중

제품을 포장하는데 제품별로 박스 당 제품 갯수가 달라 혼돈이 왔다.

간편하게 해결하기 위해 Python으로 간단하게 엑셀에서 Search 해 올 수 있는 프로그램을 만들어 보았다.
